# UI Layout Spec
# 뉴스 헤드라인 감정 분류 모니터링 대시보드

> Claude Code 참조용 문서 1/3 — UI 구성 및 레이아웃

---

## 전체 구조

```
App
├── Header (고정)
├── View 1: 모니터링 대시보드 (기본 탭)
└── View 2: 헤드라인 상세 (View 1에서 진입)
```

탭 구조: 상단 탭 2개 (`모니터링` / `헤드라인 탐색`)  
View 2는 별도 탭이 아니라 View 1 헤드라인 리스트에서 클릭 시 슬라이드-인 패널 또는 라우팅

---

## Header

```
┌──────────────────────────────────────────────────────────┐
│  [로고/타이틀]    [탭: 모니터링 | 헤드라인 탐색]          │
└──────────────────────────────────────────────────────────┘
```

- 고정(sticky) 헤더
- 타이틀: "뉴스 감정 모니터링"
- 탭 2개: 모니터링(기본 활성), 헤드라인 탐색

---

## View 1: 모니터링 대시보드

### 1-1. 컨트롤 바 (헤더 바로 아래 고정)

```
┌──────────────────────────────────────────────────────────┐
│  [6h] [1d] [1w]   |  [전체] [긍정] [중립] [부정]  |  [필터 ▼]  │
└──────────────────────────────────────────────────────────┘
```

**dt 토글** (왼쪽)
- 6h / 1d / 1w 세그먼트 버튼
- 기본값: 1d

**감정군 필터** (중앙)
- 전체 / 긍정 / 중립 / 부정 세그먼트 버튼
- 기본값: 전체
- 선택 시 Bar Chart에 표시되는 감정 라벨 필터링

**필터 패널 토글** (오른쪽)
- "필터 ▼" 버튼 클릭 시 드롭다운 패널 열림
- 활성 필터가 있으면 버튼에 개수 뱃지 표시 (예: "필터 · 3")

---

### 1-2. 필터 패널 (드롭다운)

```
┌─────────────────────────────────────┐
│  필터 선택 (최대 4개)               │
│                                     │
│  언론사                             │
│  [조선일보] [한겨레] [매일경제] ...  │
│                                     │
│  카테고리                           │
│  [정치] [경제] [사회] [문화]         │
│                                     │
│  ── 비교 단위 미리보기 ──           │
│  📌 조선일보_정치  📌 한겨레_정치   │
│                                     │
│  [초기화]              [적용]       │
└─────────────────────────────────────┘
```

**선택 규칙**
- 언론사 + 카테고리 합산 최대 4개 항목 선택 가능
- 5번째 선택 시 토스트 경고: "최대 4개까지 선택 가능합니다"

**비교 단위 미리보기** (실시간 갱신)
- 선택 조합에 따라 자동 계산된 비교 그룹을 태그로 표시
- 교차 로직은 필터링 로직 스펙(02번 문서) 참조

---

### 1-3. 메인 Bar Chart

```
┌─────────────────────────────────────────────────────┐
│  감정별 분포                    [시계열 분석 실행 →] │
│  (선택된 감정 N개)                                   │
│                                                     │
│  분노  ████████░░  ████░░░░  ─●─  (전체 평균)      │
│  불안  ██████░░░░  ███░░░░░  ─●─                   │
│  슬픔  ████░░░░░░  ██░░░░░░  ─●─                   │
│  ...                                                │
│                                                     │
│  ■ 조선_정치  ■ 한겨레_정치  ● 전체평균             │
└─────────────────────────────────────────────────────┘
```

**차트 타입**: Grouped Horizontal Bar Chart  
**Y축**: 감정 라벨 (감정군 필터 적용 후 표시)  
**X축**: Count (또는 비율 %)  
**데이터셋**: 각 필터 그룹별 색상 구분 (최대 4색)  
**오버레이**: 전체 평균을 점(●)으로 각 행에 표시  

**감정 라벨 클릭 인터랙션**
- 클릭 시 해당 감정 선택됨 (체크 표시)
- 최대 3개 선택 가능
- 3개 선택되면 우상단 "시계열 분석 실행" 버튼 활성화
- 이미 선택된 감정 클릭 시 선택 해제

---

### 1-4. 시계열 분석 팝업 (모달)

감정 1~3개 선택 후 "시계열 분석 실행" 버튼 클릭 시 전체화면 모달 오픈

```
┌────────────────────────────────────────────────────────────┐
│  시계열 분석           기간: [6h] [1d] [1w]       [✕ 닫기] │
│  선택 감정: [분노 ✕] [불안 ✕] [슬픔 ✕]                    │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  [분노]                                                    │
│  ┌──────────────────────────────────────────────────────┐  │
│  │  조선_정치 ──── 한겨레_정치 ────                     │  │
│  └──────────────────────────────────────────────────────┘  │
│                                                            │
│  [불안]                                                    │
│  ┌──────────────────────────────────────────────────────┐  │
│  │                                                      │  │
│  └──────────────────────────────────────────────────────┘  │
│                                                            │
│  [슬픔]                                                    │
│  ┌──────────────────────────────────────────────────────┐  │
│  │                                                      │  │
│  └──────────────────────────────────────────────────────┘  │
│                                                            │
│  ■ 조선_정치  ■ 한겨레_정치                               │
└────────────────────────────────────────────────────────────┘
```

**레이아웃**: 선택된 감정 수만큼 subplot (1×1 ~ 1×3 vertical stack)  
**각 subplot**: Line Chart, X축=시간, Y축=Count  
**색상**: 필터 그룹별 색상 (View 1 Bar Chart와 동일 색상 체계 유지)  
**범례**: 팝업 하단 공통 1개

---

### 1-5. 헤드라인 리스트 (Bar Chart 하단)

```
┌─────────────────────────────────────────────────────┐
│  헤드라인                          현재 필터 기준   │
│                                                     │
│  [한겨레 · 정치]  2024.01.15 14:32                 │
│  北 미사일 도발, 한반도 긴장 고조                   │
│  분노 0.91 · 불안 0.78                              │
│  ─────────────────────────────────────────────────  │
│  [조선일보 · 경제]  2024.01.15 13:10               │
│  ...                                                │
│                                                     │
│                          [더 보기 (20개씩 페이지)] │
└─────────────────────────────────────────────────────┘
```

- 현재 필터(dt + 감정군 + 필터 그룹) 기준으로 정렬: 최신순
- 언론사 + 카테고리 뱃지
- 상위 2개 감정 레이블 + confidence 표시
- 클릭 시 → View 2 (슬라이드-인 패널)

## View 2: 헤드라인 상세

View 1 헤드라인 리스트 항목 클릭 시 우측에서 슬라이드-인하는 패널 (또는 전체 페이지 전환)

```
┌──────────────────────────────────────────────┐
│  [← 목록으로]                                │
│                                              │
│  한겨레 · 정치  |  2024.01.15 14:32          │
│                                              │
│  北  [미사일]  [도발] ,  한반도  [긴장]  고조 │
│  (토큰별 Attention 하이라이팅)               │
│                                              │
│  ── 감정 분류 결과 ──                        │
│  분노   ████████████░░  0.91                 │
│  불안   ██████████░░░░  0.78                 │
│  공포   ███░░░░░░░░░░░  0.34                 │
│  ...                                         │
│                                              │
│  Evidence Tokens: "도발" · "미사일" · "긴장"  │
└──────────────────────────────────────────────┘
```

**Attention 하이라이팅**
- 토큰별 weight에 따라 배경색 강도 변화 (연파랑 → 진파랑)
- threshold 이상 토큰만 하이라이팅 (나머지는 일반 텍스트)

**감정 분류 결과**
- 전체 44개 감정 중 confidence > threshold인 것만 표시
- Horizontal bar로 confidence 시각화
- 상위 감정은 강조

**Evidence Tokens**
- Attention weight 상위 3개 토큰 텍스트로 별도 표기

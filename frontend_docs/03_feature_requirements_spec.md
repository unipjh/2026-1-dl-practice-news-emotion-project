# Feature Requirements Spec
# 뉴스 헤드라인 감정 분류 모니터링 대시보드

> Claude Code 참조용 문서 3/3 — 기능 요구사항 명세 (What & Why)

---

## 프로젝트 목적

한국 뉴스 헤드라인에 내재된 감정 편향을 Fine-grained 수준(44개 감정)으로 시각화하여,
독자가 언론의 감정적 보도 패턴을 능동적으로 파악할 수 있도록 한다.

---

## 사용자 정의

| 사용자 유형 | 핵심 Pain | 이 대시보드로 해결 |
|-------------|-----------|-------------------|
| 미디어 리터러시 독자 | "내가 읽는 언론이 얼마나 감정적인가?" | 언론사 × 카테고리 비교로 수치화 |
| 저널리즘 연구자 | "감정 패턴에 시계열 구조가 있는가?" | 시계열 팝업으로 추이 탐색 |
| 일반 뉴스 소비자 | "지금 뉴스 분위기가 어떤가?" | 전체 감정 분포 한눈 파악 |

---

## Feature 목록

### F01. dt 토글 (시간 범위 선택)

**Why**: 감정 분포는 시간 단위에 따라 완전히 다른 패턴을 보임. 단기(6h)는 이슈 반응성, 장기(1w)는 구조적 편향을 보여줌.

**What**:
- 6h / 1d / 1w 세 가지 선택 가능
- 기본값: 1d
- 토글 변경 시 Bar Chart, 헤드라인 리스트 즉시 갱신
- 시계열 팝업이 열려 있는 경우 팝업 내 기간도 동기화

**AC (Acceptance Criteria)**:
- [ ] 세그먼트 버튼 3개 렌더링, 하나만 활성 가능
- [ ] 토글 변경 시 전체 데이터 갱신 (로딩 인디케이터 표시)
- [ ] URL 쿼리 파라미터에 dt 상태 반영 (`?dt=1d`)

---

### F02. 감정군 필터 (긍정/중립/부정)

**Why**: 44개 감정을 한꺼번에 보면 인지 부하가 높음. 분석 목적에 따라 부정 감정만, 또는 긍정만 보고 싶은 니즈가 있음.

**What**:
- 전체 / 긍정 / 중립 / 부정 세그먼트 버튼
- 기본값: 전체
- 선택 시 Bar Chart에 표시되는 감정 라벨 필터링
- 중립 선택 시 3개 감정만 표시됨 (그대로 표시, 별도 처리 없음)
- 헤드라인 리스트도 해당 감정군 기준으로 필터링

**감정군 분류**: 02번 문서(필터링 로직 스펙) §2 참조

**AC**:
- [ ] 버튼 4개 렌더링, 하나만 활성
- [ ] 선택 변경 시 Bar Chart Y축 레이블 목록 갱신
- [ ] "중립" 선택 시 3개 감정만 표시되어도 UI 깨지지 않아야 함

---

### F03. 필터 패널 (언론사 × 카테고리)

**Why**: 단순 전체 분포만으로는 "어떤 언론사가 더 감정적인가"를 알 수 없음. 교차 비교가 핵심 가치.

**What**:
- 컨트롤 바 우측 "필터 ▼" 버튼으로 드롭다운 패널 토글
- 언론사 목록 + 카테고리 목록 각각 버튼 형태로 표시
- 언론사 + 카테고리 합산 최대 4개 선택
- 선택 실시간으로 "비교 단위 미리보기" 태그 갱신
- "적용" 클릭 시 필터 확정 및 데이터 갱신
- "초기화" 클릭 시 전체 필터 해제
- 활성 필터 있을 때 토글 버튼에 개수 뱃지 표시

**비교 그룹 생성 로직**: 02번 문서 §4 참조

**AC**:
- [ ] 5번째 항목 선택 시 토스트 경고 표시
- [ ] 비교 단위 미리보기 실시간 갱신
- [ ] 필터 적용 시 Bar Chart 데이터셋 색상 구분하여 업데이트
- [ ] 필터 초기화 시 전체 단일 그룹으로 복귀

---

### F04. 감정 분포 Bar Chart

**Why**: 특정 기간 동안 어떤 감정이 얼마나 많이 나타났는지 한눈에 파악하고, 그룹 간 비교를 가능하게 함.

**What**:
- Grouped Horizontal Bar Chart
- Y축: 감정군 필터 적용된 감정 레이블 (Count 내림차순 정렬)
- X축: Count (해당 기간 내 해당 감정이 threshold 이상으로 분류된 헤드라인 수)
- 데이터셋: 비교 그룹별 색상 구분 (최대 4개 그룹 = 4색)
- 전체 평균 오버레이: 각 감정 행에 ● 포인트로 전체 평균 표시
- 감정 라벨 클릭 시 해당 감정 선택/해제 (시계열 분석용)
- 선택된 감정은 라벨에 체크 표시 또는 강조

**AC**:
- [ ] 필터 그룹 없을 때: 단일 색상 Bar
- [ ] 필터 그룹 있을 때: 그룹별 색상 Grouped Bar
- [ ] 전체 평균 포인트 항상 표시 (필터와 무관하게 전체 데이터 기준)
- [ ] 감정 클릭 → 선택 상태 토글, 최대 3개
- [ ] 3개 선택 시 "시계열 분석 실행" 버튼 활성화
- [ ] 선택 감정 없을 때 버튼 비활성(disabled) 처리

---

### F05. 시계열 분석 팝업

**Why**: 특정 감정이 시간에 따라 어떻게 변화하는지, 그룹 간 차이가 시계열에서도 나타나는지 확인하는 심층 분석 기능.

**What**:
- 감정 1~3개 선택 후 "시계열 분석 실행" 버튼으로 모달 오픈
- 전체화면 모달
- 선택된 감정 수만큼 vertical subplot (Line Chart)
- 각 subplot: X축=버킷 시간, Y축=Count, 라인=비교 그룹별 색상
- 팝업 내 별도 dt 토글 (View 1 dt와 독립적으로 변경 가능)
- 선택 감정 태그 표시 (클릭 시 제거 가능)
- 닫기 버튼으로 팝업 종료 (Bar Chart 선택 상태 유지)

**AC**:
- [ ] 감정 1개: 단일 차트 (높이 크게)
- [ ] 감정 2개: 2개 차트 vertical stack
- [ ] 감정 3개: 3개 차트 vertical stack
- [ ] 모든 subplot X축 범위 동일하게 동기화
- [ ] Y축은 subplot별 독립 스케일
- [ ] 공통 범례 팝업 하단 1개
- [ ] 팝업 닫아도 Bar Chart 감정 선택 상태 초기화하지 않음

---

### F06. 헤드라인 리스트

**Why**: 집계 수치 뒤에 있는 실제 헤드라인을 탐색하고, 개별 기사의 감정 분류 결과를 확인하기 위한 진입점.

**What**:
- View 1 Bar Chart 하단에 항상 표시
- 현재 활성 필터(dt + 감정군 + 비교 그룹) 기준 최신순 정렬
- 각 항목: 언론사 뱃지 + 카테고리 뱃지 + 발행 시각 + 헤드라인 텍스트 + 상위 2개 감정
- 20개씩 페이지네이션
- 클릭 시 View 2(상세 패널) 진입

**AC**:
- [ ] 필터 변경 시 리스트 자동 갱신 (페이지 1로 리셋)
- [ ] 각 항목에 언론사 + 카테고리 뱃지 표시
- [ ] 상위 2개 감정 레이블 + confidence 수치 표시
- [ ] 로딩 중 스켈레톤 UI 표시

---

### F07. 헤드라인 상세 (View 2)

**Why**: AI가 왜 특정 감정으로 분류했는지 근거를 보여줌으로써 신뢰성을 높이고, 독자가 감정 편향을 구체적으로 인식하게 함.

**What**:
- View 1 헤드라인 클릭 시 슬라이드-인 패널 또는 페이지 전환
- 헤드라인 텍스트에 Attention 가중치 기반 하이라이팅
- 전체 감정 분류 결과 (confidence > 0.2인 감정만 표시)
- Evidence Tokens: Attention weight 상위 3개 토큰 별도 표기
- "목록으로" 버튼으로 View 1 복귀 (필터 상태 유지)

**Attention 하이라이팅 로직**: 02번 문서 §8 참조

**AC**:
- [ ] 토큰별 배경색 강도 4단계 (없음 / 연파 / 중파 / 진파)
- [ ] confidence 기준 내림차순 감정 목록
- [ ] 각 감정에 Horizontal progress bar로 confidence 시각화
- [ ] Evidence Tokens 3개 텍스트로 표시
- [ ] View 1 복귀 시 필터 상태, 스크롤 위치 유지

## 구현 우선순위

| 순위 | Feature | 이유 |
|------|---------|------|
| 1 | F04 Bar Chart + F02 감정군 필터 | 핵심 시각화, 없으면 아무것도 안 됨 |

| 3 | F06 헤드라인 리스트 | View 2 진입점 |
| 4 | F07 헤드라인 상세 | Attention 시각화, 프로젝트 차별점 |
| 5 | F03 필터 패널 | 비교 기능, 임팩트 크지만 구현 복잡 |
| 6 | F05 시계열 팝업 | 심층 분석, 마지막에 붙이기 |
